# Install expo cli
```text
npm i --global expo-cli
```

# MacOS 추가 세팅
```text
brew install watchman
```

# Create RN Project with expo
```text
expo init project-name
expo login
```

# Create RN Project without expo
```text
npm i -g create-react-native-app
create-react-native-app ProjectName
cd ProjectName
npm start
```
